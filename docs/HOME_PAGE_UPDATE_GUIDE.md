## Implementation Tips

1. **Use Squarespace Layout Blocks:**
   - For 3-column sections: Use "Layout" block with 3 columns
   - For card layouts: Use "Grid" or "Index" blocks with cards

2. **Formatting:**
   - Use **bold** for emphasis (select text ‚Üí Bold button)
   - Use proper heading levels (H1, H2, H3)
   - Add paragraph breaks for readability

3. **Buttons:**
   - Add button blocks from Squarespace's block menu
   - Set button text and link destination
   - Style as primary/secondary as needed

4. **Save Frequently:**
   - Save after completing each major section
   - Preview to check layout on desktop and mobile

5. **Waitlist Setup:**
   - Create a waitlist form page (if not exists)
   - Or link to external waitlist service
   - Update button links once waitlist is set up
