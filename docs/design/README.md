### The 5 Dimensions of Interaction Design
1. **Words** - Meaningful and simple language
2. **Visual Representations** - Images, typography, icons
3. **Physical Objects or Space** - Devices and environments
4. **Time** - Animations, videos, sounds
5. **Behavior** - Mechanisms and reactions

### Core UX Principles
- **User-Centricity** - Design with users in mind
- **Simplicity** - Keep interfaces uncluttered
- **Consistency** - Maintain uniformity
- **Feedback** - Provide immediate responses
- **Accessibility** - Design for everyone

### Mobile Design Essentials
- **Thumb-Friendly** - Reachable interactive elements
- **Performance** - Fast and responsive
- **Offline Capability** - Works without connection
- **Context Awareness** - Adapts to user situation
