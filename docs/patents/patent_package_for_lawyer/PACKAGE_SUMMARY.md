## Quick Conversion to PDF

### Easiest Method: Browser Print

1. **Open HTML file** in Safari or Chrome
2. **Press `Cmd+P`** (Print)
3. **Click "Save as PDF"**
4. **Save** with same name but .pdf extension

### Batch Conversion (macOS)

You can use this AppleScript to batch convert:

```applescript
tell application "Safari"
    repeat with htmlFile in (list of HTML files)
        open htmlFile
        delay 1
        tell application "System Events"
            keystroke "p" using command down
            delay 1
            keystroke "s" using command down
            delay 1
            keystroke (name of htmlFile without extension) & ".pdf"
            keystroke return
        end tell
        delay 2
        close front window
    end repeat
end tell
```

### Online Converter

1. Visit: https://www.ilovepdf.com/html-to-pdf
2. Upload HTML files (can do multiple at once)
3. Download PDFs
4. Save to appropriate folders
